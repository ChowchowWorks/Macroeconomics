# Macroeconomic Analysis

This repository is dedicated to practicing macroeconomic data analysis and data visualisation techniques.

Knowledge applied in this repository comes from EC2102 Macroeconomic Analysis I and DSA2101 Data Visualisation

## Topics Covered

1. **Labour Market** → [Open](labour-market/)
2. **Goods Market** → [Open](goods-market/)
3. **Asset Market** → [Open](asset-market/)

## What This Repository Contains

### Each topic folder includes:

1. Jupyter Notebook outputs (exported as HTML)
2. Data analysis and visualisations
3. Concept applications from macroeconomic theory
4. Mini-projects exploring specific economic questions

### Sources & Practice Material

Abel, Bernanke, and Croushore — Macroeconomics (7th & 11th Editions)

Additional self-initiated mini projects based on personal interest

## Objectives

* Strengthen understanding of macroeconomic concepts
* Apply theory to real or simulated data
* Improve data visualisation and analytical skills
* Build a structured portfolio of economic analysis work
